# Capturas de Pantalla

Carpeta para almacenar las capturas de pantalla de la aplicacion.

## Capturas Recomendadas

1. **dashboard.png** - Vista principal del dashboard con estadisticas generales
2. **mapa-cruces.png** - Mapa interactivo mostrando cruces semaforizados
3. **gestion-incidencias.png** - Lista de incidencias con filtros aplicados
4. **detalle-cruce.png** - Vista detallada de un cruce con perifericos
5. **mapa-incidencias.png** - Mapa de incidencias por nivel de urgencia

## Como Tomar las Capturas

1. Ejecuta la aplicacion en modo desarrollo:
   ```bash
   cd apps/frontend
   npm run dev
   ```

2. Navega a cada seccion importante

3. Toma capturas de pantalla (en Linux):
   - Presiona `PrtScn` para pantalla completa
   - O usa `Shift + PrtScn` para seleccionar area
   - O usa la herramienta Screenshot/Flameshot

4. Guarda las imagenes en esta carpeta con los nombres sugeridos

5. Asegurate de que las imagenes:
   - Tengan buena resolucion (minimo 1280x720)
   - Muestren datos de ejemplo representativos
   - No contengan informacion sensible real

## Formato de Archivo

- Formato: PNG (recomendado) o JPG
- Resolucion: 1920x1080 o superior
- Peso: Menos de 500KB cada una (optimizar si es necesario)
